Como escrever roteiros?

Você pode usar marcações mágicas que o sistema troca automaticamente pelos dados do paciente:

  • #nome - Substituído pelo 1º nome do paciente.
  • #procedimento - Tratamento de interesse.
  • #valor - Mostra o valor (ex: R$ 1.500,00).
  • #promo(30%) - Calcula um desconto no valor original (troque o 30 pelo desconto que quiser).
Destaques do GitHub (Alertas)

Para criar caixas coloridas bonitas de destaque, inicie um parágrafo novo digitando > seguido do tipo de alerta, assim:

> [!TIP]
> Essa é uma dica de ouro!
  • [!TIP] - Dica (Verde)
  • [!WARNING] - Aviso (Amarelo)
  • [!IMPORTANT] - Importante (Roxo)
  • [!CAUTION] - Cuidado (Vermelho)
